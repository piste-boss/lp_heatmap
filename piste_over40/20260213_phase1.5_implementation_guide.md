# フェーズ1.5 実装指示書 — 5-10%地点の離脱「崖」解消

## 背景と目的

### 課題
フェーズ1改修後、初動離脱率は38.6%→22.2%に改善したが、**5-10%地点で新たな離脱の崖が発生**（77.78%→28.57%、約半数が離脱）。

### 原因
1. ヒーロー直後のキャンペーン告知が「売り込み感」を与えている
2. ヒーローで判断が完結し、スクロールを続ける理由がない
3. 信頼構築がないまま価格情報を提示している

### 目標
- **5-10%地点の離脱率を50%→30%以下に改善**
- **10%地点到達率を28.57%→45%以上に引き上げ**

---

## 実施内容

### 施策A: ヒーロー直下に「信頼バッジセクション」を新設

### 施策B: キャンペーン告知を下部に移動

## 施策A: 信頼バッジセクション（詳細仕様）

### 目的
ヒーローで興味を持ったユーザーに「もう少し見てみよう」と思わせる信頼のフックを提供する。

### 配置位置
ヒーローセクションの直下、お悩み共感セクションの直上

### デザイン仕様

```
┌──────────────────────────────────────────────┐
│                                              │
│   ─── Pisteが選ばれる理由 ───                 │
│                                              │
│   🏥              📊              👥         │
│  医療機関と       継続率           40〜60代    │
│  連携            92%             専門        │
│                                              │
│   ↓ あなたに合ったプランを見る                  │
│                                              │
└──────────────────────────────────────────────┘
```

### コンテンツ詳細

3つのバッジを横並びで表示：

| バッジ | アイコン | メインテキスト | サブテキスト |
|:---|:---:|:---|:---|
| 1 | 🏥（病院アイコン） | **医療機関と連携** | 整形外科医が監修 |
| 2 | 📊（グラフアイコン） | **継続率 92%** | 無理なく続けられる |
| 3 | 👥（人物アイコン） | **40〜60代専門** | 同世代だけの安心空間 |

### スクロール促進要素

バッジの下に、下向き矢印のバウンスアニメーションを配置：
- テキスト: 「↓ あなたに合ったプランを見る」
- アニメーション: 上下にバウンス（1.5秒サイクル、無限ループ）
- スクロール開始後は非表示（IntersectionObserverで制御）

## 施策B: キャンペーン告知の移動（詳細仕様）

### 目的
ヒーロー直後の「売り込み感」を排除し、信頼構築→共感→不安解消の流れを作った上で、キャンペーンを提示する。

### 移動先
「選ばれる3つの理由」セクションの直後、「価格比較」セクションの直前

### 理由
- 3つの理由で信頼を構築した後に、お得な情報を提示する方が効果的
- 価格セクションへの自然な導線になる
- 「信頼→共感→不安解消→理由→特典→価格」の心理的フローが完成する

### 実装手順

1. 現在のキャンペーンセクション（ヒーロー直下）のHTMLブロックを**そのまま切り取り**
2. 「選ばれる3つの理由」セクションの直後に**ペースト**
3. キャンペーンセクションの上部に見出しを追加：

```html
<!-- 選ばれる3つの理由セクションの直後 -->
<section class="campaign-moved">
  <p class="campaign-lead">ここまで読んでいただいたあなたへ、特別なご案内です。</p>

  <!-- 既存のキャンペーン内容をそのまま配置 -->
  <!-- 入会金+初月0円、2月28日まで5名限定 等 -->

  <button class="cta-primary campaign-cta">体調に合わせた無料体験を予約する</button>
</section>
```

### 追加するリード文のCSS

```css
.campaign-lead {
  text-align: center;
  font-size: 14px;
  color: #666666;
  margin-bottom: 16px;
  padding: 0 20px;
}
```

## リスクと注意点

| リスク | 対策 |
|:---|:---|
| キャンペーンの移動でCV率が下がる | フローティングCTAは維持。上部でもCTAは利用可能 |
| 信頼バッジが冗長に見える | ミニマルなデザインで圧迫感を避ける |
| ページ長が微増する | 信頼バッジは高さ約100px程度。影響は最小限 |
| 移動後のモーダルの不具合 | JS内のイベントリスナーがDOM位置に依存していないか確認 |
