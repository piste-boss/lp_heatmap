# piste_lp_over40 解析レポートと修正案

## サマリー

**ターゲット**: 40代〜60代で「運動はしたいが、無理はしたくない」層  
**コンバージョン定義**: 無料体験予約の完了  
**解析期間**: 過去30日間（2026年1月11日〜2月10日）  
**主要な課題**: ファーストビューでの高い離脱率（38.6%）とユーザーの迷い

## Clarityデータ分析結果

### スクロールヒートマップの知見

![スクロールヒートマップ](/Users/ishikawasuguru/.gemini/antigravity/brain/c8a6a88b-ab4f-4a76-85df-3f988dea9b65/mobile_scroll_heatmap_top_1770731966442.png)

**クリティカルな発見**:
- **最上部（0-5%）での離脱率: 38.6%** - ページを開いた直後に4割近くが離脱
- **10%地点での累積離脱率: 85.6%** - ユーザーの9割近くが冒頭で離脱
- **50%地点到達率: わずか9.85%** - ページ中盤まで到達するのは1割以下

**推論**:
- ファーストビューの訴求力が弱い、または読み込み速度が遅い
- 「運動はしたい。でも、無理はしたくない。」というコピーは共感性が高いが、**行動喚起が弱い**
- スクロールするモチベーションを与える要素が不足

### アテンションヒートマップの知見

![アテンションヒートマップ（モバイル）](/Users/ishikawasuguru/.gemini/antigravity/brain/c8a6a88b-ab4f-4a76-85df-3f988dea9b65/mobile_attention_heatmap_1770732031470.png)

**高エンゲージメントエリア**:
- **ヒーローセクション**: 平均滞在時間 **18秒**
- **ページ中盤（40-50%地点）**: 提携クリニックやトレーナー紹介付近で **16-19秒**
- **デスクトップユーザー**: サンプル数は少ない（8 PV/30日）が、中盤セクションで **40秒以上**の長時間滞在

**推論**:
- ヒーローセクションは注目されているが、**CTAへの転換が弱い**
- 医療提携・トレーナー情報は信頼構築に貢献しているが、そこに辿り着くユーザーが少ない
- デスクトップユーザーは熟読する傾向があるが、流入数が極端に少ない（95%以上がモバイル）

### クリックヒートマップの知見

![タップヒートマップ（モバイル）](/Users/ishikawasuguru/.gemini/antigravity/brain/c8a6a88b-ab4f-4a76-85df-3f988dea9b65/mobile_tap_heatmap_1770731924233.png)

**主要クリック箇所**:
1. **FAQ項目「厳しい食事制限はありますか?」**: 20%のクリック - **最も多い**
2. **ヒーローセクションのテキスト部分**: 複数のタップ
3. **非インタラクティブな画像**: デッドクリック発生（「笑顔でカウンセリングを受ける様子」など）

**デッドクリックの詳細**:
- セッション録画で確認: ユーザーが画像をタップして詳細を確認しようとするが、**何も起こらず離脱**
- 画像に期待される行動: ギャラリー表示、拡大、関連情報への遷移など

**推論**:
- **FAQへの関心が極めて高い** → 不安払拭が購買障壁
- CTAボタンよりもテキストや画像への興味が先行 → **信頼構築不足**
- デッドクリックは機会損失 → 画像にインタラクティブ要素を追加すべき

---

### セッション録画からの行動パターン

**主要な流入元**: Facebook広告  
**典型的な行動パターン**:
1. ページ読み込み後、**数秒間ヒーローセクションを眺める**
2. **画像を1-2回タップ**（デッドクリック）
3. **すぐに離脱** または 少しスクロールしてFAQを確認後に離脱

**離脱・迷いの兆候**:
- 「入会金+初月0円」のバッジをタップするユーザーが複数確認 → **詳細を知りたいが情報が不足**
- スクロール→戻る→再度スクロールの繰り返し → **情報設計に迷いが生じている**

## 問題点の特定

### 🚨 クリティカル（優先度：最高）

#### 1. ファーストビューでの高離脱率（38.6%）
- **問題**: ページを開いた瞬間に4割が離脱。訴求力または読み込み速度に問題
- **影響**: 最大のコンバージョン損失ポイント
- **根拠**: スクロールヒートマップで0-5%地点のドロップオフが38.64%

#### 2. CTAボタンへのクリックが少ない
- **問題**: タップヒートマップでCTAボタンのクリックが見えない。FAQやテキストのタップが目立つ
- **影響**: コンバージョンへの直接導線が機能していない
- **根拠**: クリックランキングでFAQ（20%）が1位、CTAボタンが上位に入っていない

#### 3. 特典バッジのデッドクリック
- **問題**: 「入会金+初月0円」のバッジをタップするが、詳細情報がない
- **影響**: 最も魅力的なオファーの詳細を知りたいのに離脱
- **根拠**: セッション録画で複数ユーザーがバッジをタップ後に離脱

---

### ⚠️ 重要（優先度：高）

#### 4. 画像のデッドクリック
- **問題**: ユーザーが画像（カウンセリング風景など）をタップするが何も起こらない
- **影響**: ユーザーの興味を活かせていない機会損失
- **根拠**: セッション録画でデッドクリック多数確認

#### 5. FAQへの過度な依存
- **問題**: ユーザーの20%がFAQをクリック。不安が強すぎて行動に移せない
- **影響**: 信頼構築が不十分なまま最終段階に到達している証拠
- **根拠**: クリックランキングで「厳しい食事制限はありますか?」が1位

#### 6. スクロール到達率の低さ
- **問題**: 50%地点まで到達するのはわずか9.85%
- **影響**: 価格・プラン、トレーナー紹介など重要情報が見られていない
- **根拠**: スクロールヒートマップデータ

---

### 💡 改善推奨（優先度：中）

#### 7. デスクトップ流入の少なさ
- **問題**: 95%以上がモバイルユーザー。デスクトップは8 PV/30日のみ
- **影響**: 40-60代の一部層（PC利用者）を取りこぼしている可能性
- **根拠**: Clarityのデバイス別データ

#### 8. ページ中盤の高滞在時間の活用不足
- **問題**: 中盤（医療提携・トレーナー情報）で16-40秒も滞在するが、そこまで到達するのが1割未満
- **影響**: 優良コンテンツの露出不足
- **根拠**: アテンションヒートマップ

---

## 修正案（優先度順）

### 🎯 フェーズ1: Quick Wins（1週間以内）

#### 修正案1-1: 特典バッジをクリッカブルにしてモーダル表示
- **対象問題**: #3（特典バッジのデッドクリック）
- **具体的な実装**:
  - 「入会金+初月0円」バッジにクリックイベントを追加
  - モーダルウィンドウで詳細を表示:
    ```
    🎉 2月28日までの限定キャンペーン
    
    通常21,000円 → 今なら0円！
    ・入会金: 11,000円 → 0円
    ・初月会費: 10,000円 → 0円
    
    [今すぐ無料体験を予約する] ボタン
    ```
- **期待効果**: クリック率5%→10%（2倍）、CV率+15%
- **実装難易度**: 低（モーダルコンポーネント追加のみ）

---

#### 修正案1-2: ヒーローセクションのCTAを2段階に強化
- **対象問題**: #1（ファーストビューでの高離脱率）、#2（CTAクリック不足）
- **具体的な実装**:
  - **現在**: 「無料体験を予約する」のみ
  - **改善後**:
    ```html
    <div class="hero-cta-container">
      <h1>運動はしたい。でも、無理はしたくない。</h1>
      <p class="subheadline">40-60代専門・医療提携ジム</p>
      <button class="primary-cta">【無料】体験予約はこちら</button>
      <p class="social-proof">先月の体験予約: 37名 | 継続率: 92%</p>
    </div>
    ```
  - ボタンサイズを20%拡大、色を濃いオレンジ（#FF6B35）に変更
  - ボタン上に「無料」を強調
- **期待効果**: ファーストビュー離脱率 38.6%→25%、CTAクリック率+30%
- **実装難易度**: 低

---

#### 修正案1-3: FAQを冒頭（お悩み共感の直後）に移動
- **対象問題**: #5（FAQへの過度な依存）、#6（スクロール到達率の低さ）
- **具体的な実装**:
  - 現在の構成: ヒーロー → お悩み → 解決策 → ... → FAQ（最下部）
  - 改善後: ヒーロー → お悩み → **FAQ（上位3項目のみ）** → 解決策 → ...
  - 表示するFAQ:
    1. 「厳しい食事制限はありますか?」
    2. 「体力に自信がないのですが…」
    3. 「怪我をしたことがあるのですが大丈夫?」
  - 残りのFAQは「もっと見る」で展開
- **期待効果**: 不安の早期解消により、スクロール継続率+25%
- **実装難易度**: 低（セクション順序の変更）

#### 主な成果
- **中盤定着率の劇的改善**: 10%到達者のうち25%まで到達する割合が44.5%→**76.7%**

- **5-10%地点の「崖」は未解消**: 5%到達者の約65%が10%前に離脱（目標: 30%以下）

### 修正案の詳細（フェーズ2 最優先施策）

#### 信頼バッジ → 同世代の声カルーセルへの置換

**変更前**:
```
🏥 医療機関と連携 | 📊 継続率92% | 👥 40〜60代専門
```

**変更後**:
```
← 自動スライド（3秒間隔） →

「58歳、膝の手術後でも安心して通えています」
   58歳 男性 会社員

● ○ ○

↓ 同世代の方の体験を見る
```

**理由**: 静的バッジでは「見るだけ」で終わる。動的カルーセルの同世代の声は、FAQで表面化した「年齢不安」「健康不安」を先回りで解消し、スクロールを促す。
- **実装状態**: ✅ 実装済み

#### 修正案A-2: ヒーロー直下に「30秒診断」インタラクティブ要素を追加

- **対象問題**: #1（崖の未解消）、#5（能動的行動の欠如）
- **具体的な実装**:
  - カルーセルの下、スクロールヒントの前に配置:
    ```html
    <div class="quick-diagnosis">
      <p class="diagnosis-title">あなたに合った運動を10秒で診断</p>
      <p class="diagnosis-question">Q. 一番気になることは？</p>
      <div class="diagnosis-buttons">
        <button class="diagnosis-btn" data-target="#intro">体力に自信がない</button>
        <button class="diagnosis-btn" data-target="#reasons">膝や腰に不安がある</button>
        <button class="diagnosis-btn" data-target="#price">続けられるか心配</button>
      </div>
    </div>
    ```
  - 各ボタンをタップすると `data-target` 属性の値で `scrollIntoView({ behavior: 'smooth' })` を実行
  - スクロール先セクションに `id` 属性を追加: `section.intro` → `id="intro"`, `section.reasons` → `id="reasons"`, `section.price` → 既存 `id="price"`
- **理由**: ユーザーの能動的な行動を引き出し、スクロールの「理由」を自分で選ばせる
- **期待効果**: 10%到達率 24.59%→35%以上
- **実装難易度**: 中
- **実装状態**: ✅ 実装済み

#### 修正案C-1: 広告クリエイティブとLPの整合性確認（運用タスク）

- **対象問題**: #2（初動離脱率の悪化 29.51%）
- **タスク種別**: コード変更なし・運用タスク
- **具体的な実施手順**:
  1. Facebook/Instagram広告マネージャーで現在配信中のクリエイティブを確認
  2. 広告の訴求内容（コピー・画像・ターゲティング）とLPのファーストビューを比較
  3. 以下の不一致がないか検証:
     - 広告で「価格」を訴求 → LPのFVに価格表示がない
     - 広告で「医師」を訴求 → LPのFVに医療連携の言及が弱い
     - 広告のトーン（カジュアル/フォーマル）とLPのトーンの乖離
  4. ミスマッチがあれば広告またはLPのどちらかを調整
- **期待効果**: 初動離脱率 29.51%→22%以下
- **実装難易度**: 低（確認と調整のみ）
- **実装状態**: 🔲 運用タスク
