**作成日**: 2026年2月12日

## 共通デザイン要件

- **カラー**: ネイビー（#1a365d）をベース、グリーン（#06C755）をアクセント
- **トーン**: 清潔感・信頼感のあるビジネスビジュアル。堅すぎず、親しみやすさも兼備
- **NG要素**: AIロボットイラスト、漫画風キャラクター、過度な装飾
- **画像比率**: 広告 → 1:1（1080x1080px）、LPヒーロー → 16:9（1920x1080px）

## テキストオーバーレイ用ガイドライン

画像生成後、以下のテキストを別途オーバーレイする想定。画像自体にはテキストを含めない。

### 広告パターン1（実績訴求型）に重ねるテキスト:
```
★★★★★ 4.7（口コミ 53件）
「導入3週間でこの変化。」
→ 詳しい事例を見る
```

### 広告パターン2（悩み共感型）に重ねるテキスト:
```
「口コミが少ないだけで、
 来店数が30%減っている事実。」
解決策はこちら →
```

### 広告パターン3（価格訴求型）に重ねるテキスト:
```
月額2,980円で口コミが自動で増える。
導入店舗の平均: 口コミ数 +400% / 新規予約 +40%
初期費用0円キャンペーン中 →
```

## 使い方

1. **まず LP ヒーロー画像（1-A推奨）を生成** → 広告メインビジュアルと揃える基準画像にする
2. **同じトーンで広告クリエイティブ3パターンを生成** → LP ヒーローと質感を統一
3. 生成後、テキストオーバーレイは Canva や Figma で別途追加
4. 広告とLPで**同じ人物・同じ店舗の画像を使う**ことで一貫性を最大化
